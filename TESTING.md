# Chameleon — 使用与测试指南

> V1 色板管理工具 — 创建色板、生成辅助配色、管理色彩方案

---

## 快速开始

### 1. 访问地址

- **Vercel 生产环境（推荐）：** 部署后自动可用
- **本地开发：** `pnpm dev` → http://localhost:5173

### 2. 用户流程（完整体验路线）

```
首页 → 新建色板 → 选主色 → 看候选色 → 分配角色 → 保存 → 回到首页看卡片
```

---

## 测试步骤

### 步骤 1：首页 — 空态引导

| 检查项   | 预期结果                                            |
| -------- | --------------------------------------------------- |
| 打开首页 | 看到英雄区 "让颜色先被看见。" 标题 + 渐变氛围球装饰 |
| 空状态   | 显示 "还没有色板" + "创建色板" 按钮                 |
| 设计风格 | 米白底色、衬线字体标题、大圆角卡片                  |

### 步骤 2：新建色板

| 操作                               | 预期结果                                                          |
| ---------------------------------- | ----------------------------------------------------------------- |
| 点击 "创建色板" 或 "新建色板" 按钮 | 跳转到 `/palette/new`                                             |
| 页面布局                           | 顶部返回 + 保存按钮，中间从左到右：主色区 → 候选色 → 色板角色侧栏 |
| 色板名称输入                       | 在顶部大标题区直接输入名称，回车可换行                            |

### 步骤 3：选取主色

| 操作                        | 预期结果                           |
| --------------------------- | ---------------------------------- |
| 点击 color picker           | 弹出系统颜色选择器，选择后实时更新 |
| 在 HEX 输入框输入 `#ff0000` | 主色变为红色，所有候选色实时更新   |
| 输入非法值                  | 不会更新（有校验）                 |

### 步骤 4：辅助配色工具

| 候选色分组   | 说明               | 检查数量   |
| ------------ | ------------------ | ---------- |
| **补色**     | 主色色环 180° 对面 | 1 个候选色 |
| **分裂补色** | 主色 ±150°         | 2 个候选色 |
| **类似色**   | 主色 ±30° 相邻色   | 3 个候选色 |
| **三元色**   | 色环 120° 均分     | 3 个候选色 |
| **单色阶**   | 同色相明度 5 级    | 5 个候选色 |

> 验证：切换主色（如 `#ff0000` → `#6366f1` → `#00ff00`），所有候选色分组应实时更新。

### 步骤 5：候选色一键应用到角色

| 操作                    | 预期结果                                     |
| ----------------------- | -------------------------------------------- |
| 点击任意候选色块        | 底部 Sheet 弹出，显示颜色预览 + 5 个角色按钮 |
| 点击 Sheet 外的灰色遮罩 | 关闭 Sheet                                   |
| 点击 "背景色"           | 该候选色自动填入 background，Sheet 关闭      |
| 侧栏色板角色区          | 对应的角色颜色已更新                         |

### 步骤 6：手动微调配色

| 操作                              | 预期结果                                                    |
| --------------------------------- | ----------------------------------------------------------- |
| 在右侧色板角色区点击 color picker | 可单独修改每个角色的颜色                                    |
| 在 HEX 输入框修改                 | 实时更新对应角色色块                                        |
| 5 个角色                          | primary / surface / background / text / accent 均可独立编辑 |

### 步骤 7：保存色板

| 操作            | 预期结果                           |
| --------------- | ---------------------------------- |
| 点击 "保存色板" | 跳转回首页                         |
| 首页显示        | 刚才创建的色板以卡片形式展示       |
| 色板卡片        | 显示名称 + 描述 + 5 个角色色块预览 |

### 步骤 8：多次创建验证

| 操作                            | 预期结果                              |
| ------------------------------- | ------------------------------------- |
| 重复步骤 2-7，创建 3 个不同色板 | 首页网格显示 3 张卡片                 |
| 刷新页面                        | 色板列表不丢失（localStorage 持久化） |
| 删除 localStorage               | 回到空状态                            |

## 已知限制（V1）

- 色板仅存储在浏览器 localStorage，清除浏览器数据会丢失
- 没有删除/编辑已有色板功能
- 没有模板预览（微信/X 预览尚未实现）
- 不支持导出色板
- 暗色模式尚未完全适配
